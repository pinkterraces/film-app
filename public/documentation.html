Documentation

Purpose

This app allows users to create a profile and save a list of films they're interested in, as well as browse and find information on similar films.

The API uses Node.js to interface with a database.

API Endpoints

Get list of all movies
Purpose / Business Logic Retrieve all films in database
Resource URL /movies
Parameters None
HTTP Method GET
Request data format UTF-8 encoded URL
Response data format JSON object with list of all films and their data

Create New User
Purpose / Business Logic Create a new user account in the database
Resource URL /users
Parameters None
HTTP Method POST
Request data format JOSN object with mandatory key and value for "name"
Response data format JSON with user data + newly created user ID
Examples
Example Request 
{ "name": "Mary Smith" }
Example Response 
          {
            "name": "Mary Smith",
            "id": "7bcfdda0-ea20-486c-b8e4-77dcbf4b0625"
        }
Update User Name
Purpose / Business Logic Update user name in the database
Resource URL /users/[Id]
Parameters Id
HTTP Method PUT
Request data format URL with user id and JSON providing the user name with key "name"
Response data format Text confirmation with new username
Examples
Example Request /users/1
{ "name": "Mary Brown" }
Example Response 
          {
            "id": 2,
            "name": "Mary Brown",
            "favouriteMovies": [
                "The Fly"
            ]
        }
Delete User
Purpose / Business Logic Delete user from the database
Resource URL /users/[Id]
Parameters Id
HTTP Method DELETE
Request data format UTF-8 encoded URL with user id
Response data format Text confirmation with username that was removed
Examples
Example Request /users/2
Example Response "User with ID: 2 has been deleted"
Add to favourites
Purpose / Business Logic Adds movie to users list of favourite movies
Resource URL /users/[Id]/favourites/[movieTitle]
Parameters Id
Movie name in UTF-8 format
HTTP Method POST
Request data format UTF-8 encoded URL with user id and title of movie
Response data format Text confirmation with name of list + film title added
Examples
Example Request /users/2/Movie%20Title
Example Response "The Swarm has been added to favourites for user with ID: 3"
Delete from favourites
Purpose / Business Logic Remvoes a movie from users list of favourite movies
Resource URL /users/[Id]/favourites/[movieTitle]
Parameters Id
Movie name in UTF-8 format
HTTP Method DELETE
Request data format UTF-8 encoded URL with user id and title of movie
Response data format Text confirmation with name of list + film title removed
Examples
Example Request /users/2/Movie%20Title
Example Response "The Swarm has been removed from user's (ID: 3) favourites"
